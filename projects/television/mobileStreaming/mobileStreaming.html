Mobile Streaming

Necessary Terms

In my experience of mobile streaming, I think the quality and the playback system needs a lot of improvements to become one of the everyday tools of a user.

VGA (Video Graphics Array): Size in Pixels : 640 x 480

WXGA (Wide Extended Graphics Array): Size in Pixels : 1366 x 768

CIF (Common Intermediate Format) : Size in Pixels : 352 x 240

QCIF (Quarter Intermediate Format): Size in Pixels : 176 x 144

XGA (Extended Graphics Array) : Size in Pixels : 1024 x 768

SXGA (Super Extended Graphics Array): Size in Pixels : 1280 x 1024

UXGA (Ultra Extended Graphics Array): Size in Pixels : 1600 x 1200

QVGA (Quad Extended Graphics Array): Size in Pixels : 2048 x 1536
